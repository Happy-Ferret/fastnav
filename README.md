#Global Action
An add-on that allows you access any action with a single a place

##Actions implemented
- About Fast Navigation
- Close Tab
- Go Back
- Go Forward
- New Tab
- Preferences
- Search in Bookmarks
- Search in History
- Switch between opened Tabs
